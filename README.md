Nuxt & DRF Collaboration
===


1. DRF(Django REST Framework)

```bash
$ cd backend

# Install dependencies
$ pip install -r requirements.txt

# Run
$ python manage.py migrate
$ python manage.py runserver 0.0.0.0:8000
```

2. Nuxt

```bash
$ cd frontend

# Install dependencies
$ npm install

# Run
$ npm run dev
```
